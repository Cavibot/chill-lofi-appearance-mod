# Required DLL References for Building

⚠️ DO NOT upload these files to GitHub - they are copyrighted!

This folder should contain the following DLL files copied from your game installation.

## BepInEx Dependencies (from BepInEx\core\)

Copy from: `[Game]\BepInEx\core\`

- 0Harmony.dll
- BepInEx.dll

## Game Dependencies (from Game's Managed folder)

Copy from: `[Game]\Chill With You_Data\Managed\`

- Assembly-CSharp.dll

## Unity Engine Dependencies (from Game's Managed folder)

Copy from: `[Game]\Chill With You_Data\Managed\`

- UnityEngine.dll
- UnityEngine.AssetBundleModule.dll
- UnityEngine.CoreModule.dll
- UnityEngine.IMGUIModule.dll
- UnityEngine.InputLegacyModule.dll
- UnityEngine.InputModule.dll
